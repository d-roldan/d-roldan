## Sobre mi

Soy David Roldan, fundador de **DRAIT**, una empresa enfocada en soluciones de software industrial, IoT y trazabilidad para operaciones reales de planta.

Trabajo en la interseccion entre el mundo industrial y el software moderno: conectando sensores, PLCs, SCADA, gateways edge, bases de datos, dashboards y aplicaciones web para que la informacion operativa sea visible, confiable y accionable.

Mi objetivo es construir sistemas que puedan implementarse en entornos industriales reales: robustos, mantenibles, documentados y preparados para funcionar on-premise, en edge o en cloud segun la necesidad operativa.

---

## Que hacemos en DRAIT

- Software industrial a medida para talleres, pymes industriales y plantas.
- Trazabilidad de produccion por lote, pieza, operacion y orden de trabajo.
- Dashboards y KPIs en tiempo real para OEE, calidad, backlog y disponibilidad.
- Integracion con PLCs, SCADA, OPC-UA, Modbus, MQTT y sistemas existentes.
- IoT industrial con ESP32, Raspberry Pi, LoRa, sensores y gateways edge.
- Infraestructura OT/IT: Docker, Linux, redes segmentadas, backups y monitoreo.
- Automatizacion entre ERP, MES, calidad, mantenimiento y reportes.

## Como trabajo

- Diseno soluciones desde el problema operativo, no desde la herramienta.
- Integro campo, edge, servidor, base de datos, dashboard y reportes.
- Prefiero stacks abiertos, reproducibles y mantenibles.
- Priorizo sistemas simples de operar, pero preparados para escalar.
- Documento arquitectura, decisiones tecnicas y despliegues para que el proyecto sea sostenible.
